Procurement of Components
Learning Objectives

Follow a basic procedure for procurement, that includes:

  • Matching the Bill of Material (BOM) to the design of the system
  • Match technical specification of an item to the requirement as specified in the BOM
  • Use the decision table provided to evaluate different available choices of specified components and pick the appropriate one
Duration:20 hours including real purchasing activity

Introduction

Any engineer or technician is always likely to run into the need for procuring items. It is essential to learn the skills to recognize the right components based on requirements.

  • Procurement Process

 

Conduct Activities

  1. Download BOMs available online and match with SLD
  2. Verify if actual components in the lab match with a BOM. Note all variations in a log sheet
  3. Match QC certificates with available components. Match and verify deviations
  4. Collect and file all documentation needed for procurement

Introduction

It is very common that you will not find the exact components that have been specified in the BOM. During such times, you may have to evaluate alternatives. In this session, we will look at the options you have for purchasing each component
