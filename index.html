Welcome


Climate Trainers Training Sessions

Session 1

Learn basics electrical term and working principle of solar cell

  • Session 1.1

    Light To Electricity

    Learning Objectives :

    1. Visualize how light energy generates electricity
    2. Learn how to connect the wires to power an LED using a PV module
    3. Visualize the working principle of a solar cell

  • Session 1.2

    Electrical Basics

    Learning Objectives :

    1. Explain the concept of voltage, current, power and energy.
    2. Create series and parallel configuration with loads and predict the voltage and current.
    3. Explain the function of electrical components like fuses, capacitors and circuit breakers
    4. Create simple electrical circuits by reading circuit diagrams

Session 2

Input energy for solar is the sun radiation. Learn basics of radiation in electrical terms.

  • Session 2.1

    Solar Energy and Radiation

    Learning Objectives :

    1. Distinguish between heat and light energy and solar thermal and solar PV technology.
    2. Distinguish between direct and diffuse radiation and how they are measured

  • Session 2.2

    Solar PV Output

    Learning Objectives :

    1. Explain what is voltage, current and power.
    2. Predict how solar PV output varies with load.

  • Session 2.3

    Solar PV Module Specifications

    Learning Objectives :

    1. Read and interpret the specifications of a PV module.
    2. Predict the output of solar PV modules in series and parallel configurations.
    3. Describe the different PV module technologies and their performance.

  • Session 2.4

    Solar PV Output Variation

    Learning Objectives :

    1. Estimate how PV output varies with tilt and shading

Session 3

Learn how to select system components and system type of solar PV. Learn load and site analysis techniques.

  • Session 3.1

    Solar PV System Components

    Learning Objectives :

    1. Identify all components in a DC and AC solar PV system.
    2. Explain the functioning of each component of a solar system.
    3. Explain how AC and DC solar systems differ.
    4. Explain energy conversion and loss in a PV system.

  • Session 3.2

    Solar PV System Types

    Learning Objectives :

    1. Distinguish the 3 types of solar PV systems by components used and behavior of the systems
    2. Predict how each system will behave under scenarios such as overloading, battery charging or power cuts

  • Session 3.3

    Customer Needs and Solutions

    Learning Objectives :

    1. Map key customer need to a solar solution that needs to be provided.
    2. Create simple block diagrams to represent different solar solutions.

  • Session 3.4

    Load Assessment

    Learning Objectives :

    1. Distinguish between load and energy requirements.
    2. Gather energy and electrical load use information for a building.
    3. Create a load profile that shows peak and average load.

  • Session 3.5

    Site Assessment

    Learning Objectives :

    1. Create a sun path diagram and perform shading analysis.
    2. Assess the site for suitability of a solar PV system.
    3. Mark potential areas where the solar array and other components could be installed.

Session 4

Take a deep dive into PV system components

  • Session 4.1

    Batteries

    Learning Objectives :

    1. Explain how a battery works.
    2. Explain how batteries are rated.
    3. Predict the voltage, current and energy from a battery bank in different configurations.
    4. Describe the charging and discharging characteristics of batteries.
    5. Explain the differences and trade-offs between different types of batteries.
    6. Choose the right type of battery for a given application.

  • Session 4.2

    Inverters and Charge Controllers

    Learning Objectives :

    1. Explain AC and DC current.
    2. Explain the purpose and working principle of charge controllers.
    3. Explain the working principle of inverters.
    4. Identify the different types of inverters used in a solar system.

  • Session 4.3

    Wiring in PV Systems

    Learning Objectives :

    1. Recognize the different types and sizes of wires and cables used in solar PV systems.
    2. Recognize the different types and sizes of protection devices used in a solar PV system.
    3. Use a decision table to pick the right sizes for wires and protection devices for a PV system configuration.

Session 5

Learn how to size a solar PV system.

  • Session 5.1

    Using a Single Line Diagram

    Learning Objectives :

    1. Read and interpret a single line diagram of different PV configurations.

  • Session 5.2

    Solar PV System Sizing

    Learning Objectives :

    1. Pick the right system component types based on customer requirements.
    2. Use load and energy requirements data from a site survey to size the system components.

Session 6

Learn all about installation and inspection of a solar PV plant.

  • Session 6.1

    Preparing for Installation

    Learning Objectives :

    1. Create a step-by-step installation plan for a 1 kilowatt system.
    2. Identify and take all safety precautions before starting an installation.
    3. Identify and gather all the tools and materials needed for installation.

  • Session 6.2

    Procurement of Components

    Learning Objectives :

    1. Distinguish the 3 types of solar PV systems by components used and behavior of the systems.
    2. Match technical specification of an item to the requirement as specified in the BOM.
    3. Use the decision table provided to evaluate different available choices of specified components and pick the appropriate one.

  • Session 6.3

    Civil and Mechanical Work

    Learning Objectives :

    1. Identify the factors that go into designing the mounting structure.
    2. Identify the different types of tracking mechanisms used in an installation.
    3. Supervise the construction of mounting for a small scale rooftop PV system.

  • Session 6.4

    DC Side Wiring

    Learning Objectives :

    1. Take safety measures before doing the wiring.
    2. Identify the key components and steps needed for DC side wiring.
    3. Explain why DC side wiring is different from AC side wiring.
    4. Perform DC side wiring of a solar PV system.
    5. Identify and explain the use of blocking and bypass diodes.

  • Session 6.5

    AC Side Wiring

    Learning Objectives :

    1. Take safety measures before doing wiring.
    2. Identify the key components and steps needed for AC side wiring.
    3. Map the existing building wiring.
    4. Perform AC side wiring of a solar PV system in the correct sequence by referring to the Standard Task Procedure.

  • Session 6.6

    Visual Quality Inspection

    Learning Objectives :

    1. Inspect the system and identify quality issues.

  • Session 6.7

    System Testing and Commissioning

    Learning Objectives :

    1. Perform specific tests to test the system before commissioning.
    2. Orient the customer with adequate information and knowledge.

Session 7

Learn about manintenance and monitoring of a PV system.

  • Session 7.1

    Preventive Maintenance

    Learning Objectives :

    1. Explain the areas where maintenance is frequently required and the consequences of not attending to them.
    2. Use a checklist to inspect the system and detect components requiring attention.
    3. Perform maintenance tasks from manuals.

  • Session 7.2

    System Monitoring

    Learning Objectives :

    1. Perform tests to determine the efficiency of the PV system.
    2. Explain the ways in which automated monitoring of system performance could be useful.
    3. Explain what sensors should be used and where in the system they should be attached.
    4. Interpret performance monitoring data from automated sensors.

  • Session 7.3

    Wiring in PV Systems

    Learning Objectives :

    1. Follow a systematic procedure to troubleshoot the system by reading a troubleshooting table.
    2. Work through basic troubleshooting steps for inverters, batteries, PV modules and wiring.
    3. Describe problems that typically occur in all components and their causes.
    3. Fill a job card to ensure the problem and solution are documented.
