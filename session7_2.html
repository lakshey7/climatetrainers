System Monitoring
Learning Objectives
  • Perform tests to determine the efficiency of the PV system
  • Explain the ways in which automated monitoring of system performance could be useful
  • Explain what sensors should be used and where in the system they should be attached
  • Interpret performance monitoring data from automated sensors
